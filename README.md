# Junkai

Junkai は、複数形式のファイルを横断して検索できるデスクトップ向け検索ツールです。

## 特徴

- テキストファイルの検索に対応
- Excel、Word、PowerPoint ファイルも横断検索可能
- インストール版とポータブル版を提供

## ダウンロード

最新版は以下からダウンロードできます。

https://github.com/motomuuu/Junkai/releases/latest

- 通常はこちら  
  `Junkai_Setup_vx.x.x.exe`

- インストールせずに使いたい場合  
  `Junkai_Portable_vx.x.x.zip`


## インストール方法

1. `Junkai_Setup_vx.x.x.exe` をダウンロードします。
2. ダウンロードしたファイルを実行します。
3. 画面の案内に従ってインストールします。
4. スタートメニュー、またはデスクトップショートカットから Junkai を起動します。

## ポータブル版の使い方

1. `Junkai_Portable_vx.x.x.zip` をダウンロードします。
2. ZIP ファイルを任意の場所に展開します。
3. 展開したフォルダ内の実行ファイルから Junkai を起動します。
